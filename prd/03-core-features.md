# Core Features

## Must-Have (MVP)

### 1. Hero / Landing Section
- Name, title/tagline, and a short bio
- Call-to-action buttons (e.g., "View Work", "Contact Me", "Download Resume")
- Professional photo or avatar
- Animated background (particle field / aurora)
- Social media icon links (GitHub, LinkedIn, X)

### 3. Tech Stack / Skills Section
- Categorized skill cards (Frontend, Backend, DevOps, AI, etc.)
- Proficiency badges or progress indicators per skill
- Tool/framework icons with tooltips
- Grouped by: Languages, Frontend, Backend, Databases, Cloud, AI/ML

### 4. Projects Section
- Project cards with title, description, tech stack used
- Links to live demo and GitHub repo
- Filterable by category or tech
- Featured / pinned project highlight
- Project thumbnails or screenshots

### 5. Experience / Timeline Section
- Work experience cards with company, role, and duration
- Education timeline (BS Computer Science — Final Year)
- Key achievements and responsibilities per role
- Vertical or horizontal timeline layout

### 6. Testimonials / Recommendations Section
- Testimonial cards with name, role, company, and photo
- Carousel or grid layout
- Star rating or endorsement badges
- LinkedIn recommendation links

### 7. Open Source / GitHub Section
- GitHub contribution graph or stats embed
- Pinned repositories showcase
- Total stars, forks, and contributions
- Link to GitHub profile

### 9. Contact Section
- Contact form (name, email, message) with validation
- Links to social profiles (GitHub, LinkedIn, X/Twitter, Gmail)
- Location / timezone info
- Response time expectation ("I'll reply within 24 hours")

### 10. Navigation
- Sticky/fixed top navbar with smooth scroll
- Mobile-responsive hamburger menu
- Active section highlight on scroll
- Logo / name branding on the left

### 11. Footer
- Copyright notice
- Quick links (About, Projects, Contact)
- Social media icons
- "Built with Next.js & Tailwind" credit
